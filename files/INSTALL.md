# How to Install This Browser Extension

## For Chrome, Edge, or Brave:

1. Open your browser
2. Go to extensions page:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
   - Brave: `brave://extensions`

3. Enable "Developer mode" (toggle switch in top right)

4. Click "Load unpacked"

5. Select the `resume-extension` folder

6. Done! The extension icon (ðŸ“„) should appear in your toolbar

## For Firefox:

1. Open Firefox
2. Go to `about:debugging#/runtime/this-firefox`
3. Click "Load Temporary Add-on"
4. Navigate to this folder and select `manifest.json`
